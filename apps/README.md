# apps/

| Folder | Service | Phase |
|---|---|---|
| `apps/frontend/` | React + Vite SPA (prime-frontend) | Phase 8+ |
| `apps/backend/` | Fastify + TypeScript API (prime-backend) | Phase 6+ |

The Dockerfiles are already in place. Application source code is added during Phase 6 (foundation) and Phase 8 (frontend).

Do not add application code here until Phase 4 is approved and the Phase 5 prototype is signed off.
